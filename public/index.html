Step 1 — Browser Server URL
Puppeteer server URL (your headless browser backend)

Deploy server.js to Railway (free tier). Show deployment guide ▾

Deploy to Railway (free, ~3 min):
  1. Go to railway.app → sign up free with GitHub
  2. Click New Project → Deploy from GitHub repo
  3. Push these files to a GitHub repo: server.js, package.json, nixpacks.toml
  4. Railway auto-detects Node.js and deploys automatically
  5. Go to your project → Settings → Networking → Generate Domain
  6. Copy the URL (e.g. https://cdn-detector.up.railway.app)
  7. Paste it in the field above

Railway free tier: 500 hours/month — plenty for scanning thousands of companies.

Step 2 — Google account

Needed to read and write your Google Sheet. Show OAuth setup ▾

One-time Google OAuth setup (~3 min):
  1. Go to console.cloud.google.com → create a project
  2. APIs & Services → Library → enable Google Sheets API
  3. Credentials → Create Credentials → OAuth 2.0 Client ID → Web application
  4. Add your site URL to Authorised JavaScript origins
  5. Copy the Client ID and paste below
  6. OAuth consent screen → Test users → add your Gmail
